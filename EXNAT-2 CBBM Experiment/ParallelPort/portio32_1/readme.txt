Winford Engineering PortIO32 Overview

PortIO32 is a simple port I/O driver and library provided by Winford 
Engineering.  Please see the included documentation for a detailed 
description of the functions available for use, the installation 
procedures, how to use the library from a programming language, and 
other information.


The following directories are included in this distribution:
==============================
documentation - Describes the installation and use of PortIO32.
                Includes a reference listing of the functions available.

win - Drivers and programming language support files for 
      Microsoft Windows computers

linux - Programming language support files for use on Linux

samples - Sample applications in various languages that demonstrate
          how to use PortIO32
